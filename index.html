📸 CÓMO SUBIR LAS FOTOS
  1. Saca las fotos con el celular y pásalas al computador
  2. Entra a github.com con tu cuenta PaldeaGames
  3. Abre tu repositorio de imágenes
  4. Add file → Upload files y arrastra todas las fotos juntas
  5. Click en Commit changes
  6. Abre cada foto → clic derecho sobre la imagen → Copiar dirección de imagen
  7. Pega las URLs aquí abajo, una por línea
✅ La URL correcta se ve así:
https://github.com/PaldeaGames/.../1.jpg?raw=true
o también raw.githubusercontent.com/...
💡 Si las nombras 1.jpg, 2.jpg, 3.jpg… copias la primera URL y solo cambias el número para el resto.
